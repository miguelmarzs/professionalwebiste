College Consulting Project

The Goal Diggers, composed of 3 members, designed and created a prototype for a system for the Media Technology Services at Desert View Community College. This report includes the Context and System Diagrams along with the Lower-Level Data Flow Diagrams for the system. Additionally, this report includes the Entity Relationship Diagram, Data Model, Process Specifications, Interface and Output Design elements, as well as a Prototype.
